Mapa Folium

Geradoras Eólicas e Fotovoltaicas do Brasil

Mapa interativo que fornece informações das geradoras eólicas e fotovoltaicas localizadas no Brasil. O mapa foi gerado por meio das bibliotecas python Folium e Geopandas, e a aplicação desenvolvida por meio do microframework Flask e ferramentas básicas de Front-End (HTML, CSS e Javascript)

Informações sobre o projeto

Projeto desenvolvido para a apresentação da ferramenta Flask para o desenvolvimento de dashboards e aplicações com Python.
